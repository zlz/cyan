本网站为技术验证之用，正在开发中。采用自适应布局，主要使用了以下技术

angularjs

采用angular 1.x 做前端主要架构，并加入了相关组件。
angular-ui-router；
angular-animate；
angular-touch；
ui-bootstrap

sass

样式表使用sass格式，在gulp中使用gulp-sass编译；使用gulp-autoprefixer做多浏览器兼容；最后使用gulp-csso做minify。

bootstrap

使用了它的一些样式。如自适应布局等，达到一套代码多终端浏览。

jquery

采用了jQuery 2.x版本，彻底抛弃IE9以下老旧浏览器，老版本浏览器引导用户到浏览器下载网站下载新版本浏览器。

zSlide

自己写的幻灯片插件。支持响应式布局，分辨率改变时会相应改变图片大小，原来写的时候是支持IE6的，现在结合了angular-touch等功能。

iconfont.cn

使用iconfont.cn里的矢量图标做成一个库，减少了页面大小。

animate.css

做效果。

gulp

编译时用gulp做任务管理，
asset文件的拷贝；dist清理；gulp-changed监控文件变化；gulp-filter过滤scss；gulp-rename重命名生成的文件；gulp-htmlmin优化html文件；

webpack

webpack放到了gulp里，使用它把页面js和资源js分别打包；使用babel-loader做ES6兼容；再使用uglifyjs-webpack-plugin做js压缩。

nodejs

nodejs配合express，gulp，bower，justreq使用。并使用package.json做服务器管理。

express

使用express搭建前端代码运行环境，使用http-proxy-middleware作为代理，解决前端代码的跨域问题，方便调试。

MongoDB&Mongoose

MongoDB提供数据存储，使用Mongoose进行数据增删改查。

eslint

统一代码格式，方便团队开发。

git

本网站发布在http://www.github.com/zlz/cyan。MIT

外源api

《新闻》栏目采用了外源api的数据。解决跨域的方法是在vps上建个node环境，用代理和cors方法把数据转一下。外源免费数据有点慢...
